# Replit Prompt: Mac Network Sharing & Screen Share App

## Project Overview

Build a native macOS application called **"NetShare"** (or similar) using SwiftUI that provides a beautiful, unified interface for:

1. **Enabling and configuring screen sharing** on the local Mac
2. **Enabling and configuring SMB file sharing**
3. **Discovering other Macs on the local network** via Bonjour/mDNS
4. **Initiating screen sharing sessions** with discovered computers
5. **Managing trusted devices** for quick access

This is a Mac-to-Mac, local network only solution with security as a priority.

---

## Technical Requirements

### Platform & Frameworks
- **macOS 13.0+ (Ventura)** minimum deployment target
- **SwiftUI** for all UI (no AppKit except where absolutely necessary)
- **Combine** for reactive data flow
- **Network.framework** for Bonjour/mDNS discovery
- **ServiceManagement** for system service configuration
- **SystemConfiguration** for network status monitoring

### Architecture
- **MVVM** with clear separation of concerns
- Dedicated service classes for:
  - `NetworkDiscoveryService` - Bonjour browsing for `_rfb._tcp` (VNC) and `_smb._tcp` services
  - `SharingConfigurationService` - Managing local sharing settings
  - `ScreenShareConnectionService` - Initiating VNC connections
  - `SecurityService` - Keychain storage, trusted device management

### 3. Network Discovery

Implement robust Bonjour service discovery:

```swift
// Services to browse for:
let services = [
    "_rfb._tcp.",      // VNC/Screen Sharing
    "_smb._tcp.",      // SMB File Sharing  
    "_afpovertcp._tcp.", // AFP (legacy)
    "_device-info._tcp." // Device metadata
]
```

**Discovery Service Requirements:**
- Continuous background scanning with configurable intervals
- Resolve hostnames to IP addresses
- Detect when devices come online/go offline
- Cache device info (name, IP, last seen, services available)
- Handle network changes gracefully (WiFi switch, etc.)

### 4. Screen Share Connection

When user clicks "Share Screen" on a discovered device:

```swift
// Option 1: Use macOS Screen Sharing app
let url = URL(string: "vnc://\(ipAddress)")!
NSWorkspace.shared.open(url)

// Option 2: Use open command with screensharing URL scheme  
// vnc://username@hostname

// Option 3: For more control, launch Screen Sharing.app directly
let screenSharingPath = "/System/Library/CoreServices/Applications/Screen Sharing.app"
```

**Connection Flow:**
1. User clicks device → Connection sheet appears
2. Show connection options:
   - "Connect as: [current username]" 
   - "Request permission to control" vs "Observe only"
   - "Save credentials for this device" checkbox
3. Initiate connection
4. Handle authentication (offer to save to Keychain)

### Bonjour Discovery

```swift
import Network

class NetworkDiscoveryService: ObservableObject {
    @Published var discoveredDevices: [DiscoveredDevice] = []
    
    private var browser: NWBrowser?
    
    func startDiscovery() {
        let parameters = NWParameters()
        parameters.includePeerToPeer = true
        
        browser = NWBrowser(for: .bonjour(type: "_rfb._tcp.", domain: "local."), using: parameters)
        
        browser?.browseResultsChangedHandler = { [weak self] results, changes in
            DispatchQueue.main.async {
                self?.handleDiscoveryResults(results)
            }
        }
        
        browser?.start(queue: .main)
    }
    
    private func handleDiscoveryResults(_ results: Set<NWBrowser.Result>) {
        // Process and update discoveredDevices
    }
}
```

### Check Sharing Status
```swift
func isScreenSharingEnabled() -> Bool {
    let task = Process()
    task.executableURL = URL(fileURLWithPath: "/bin/launchctl")
    task.arguments = ["print", "system/com.apple.screensharing"]
    
    let pipe = Pipe()
    task.standardOutput = pipe
    task.standardError = pipe
    
    do {
        try task.run()
        task.waitUntilExit()
        
        let data = pipe.fileHandleForReading.readDataToEndOfFile()
        let output = String(data: data, encoding: .utf8) ?? ""
        return output.contains("state = running")
    } catch {
        return false
    }
}
```
